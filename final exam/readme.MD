# 📊 DMart Sales Data Dashboard (Excel Project)

An interactive Excel dashboard designed to analyze **DMart's** sales performance, customer types, product categories, and key business metrics in a clear and visual way.

This project demonstrates how raw DMart sales data can be transformed into meaningful insights using Microsoft Excel tools such as **Pivot Tables**, **Pivot Charts**, **Slicers**, and **Conditional Formatting**.

---

## 🚀 Project Overview

The **DMart Sales Data Dashboard** provides a centralized view of DMart's retail business performance across **2,000 transactions** spanning from **January 2023 to January 2025**.

It allows users to quickly analyze:

- 💰 **Total Sales Revenue**
- 📦 **Total Orders Placed**
- 📈 **Sales Trends Over Time**
- 🛍️ **Product Category Contribution**
- 👤 **Customer Type Performance**
- 📊 **Sales Category Distribution** (Low / Medium / High)

The dashboard is fully interactive, enabling users to filter and explore the data dynamically using slicers.

## 🧩 Dashboard Structure

### 🔹 1️⃣ Raw Data Sheet
The foundation of the entire dashboard. Contains **1,999 rows** of transactional sales records with the following fields:

| Column | Description |
|---|---|
| `Date` | Transaction date (Jan 2023 – Jan 2025) |
| `Time` | Time of transaction |
| `CustomerType` | Member / Premium / Regular |
| `ProductCategory` | Product category of the sale |
| `UnitPrice` | Price per unit |
| `Quantity` | Number of units sold |
| `Total` | Total sale amount |
| `FullName` | Customer full name |

---

### 🔹 2️⃣ Data Cleaning Sheet
A cleaned and enriched version of the raw data with additional derived columns:

- **SALES CATEGORY** — Classifies each transaction as `Low Sales`, `Medium Sales`, or `High Sales`
- **DISCOUNT ELIGIBILITY** — Flags discount eligibility for Premium members only
- **Computed KPIs** — Total Sales: **₹1,546,667.07** calculated directly in this sheet

---

### 🔹 3️⃣ Pivot Table Sheet
Houses all the pivot tables that power the dashboard charts:

- 📊 **Sales by Product Category** — Breakdown of revenue across all 5 categories
- 📅 **Sales by Year** — Year-over-year comparison (2023, 2024, 2025)
- Additional aggregations used for KPI cards and trend analysis

---

### 🔹 4️⃣ Charts Sheet
Contains all the pivot charts linked to the pivot tables, including:

- 📊 **Category Wise Sales** (Bar / Column Chart)
- 📈 **Sales Trend Over Time** (Line Chart)
- 🥧 **Customer Type Distribution** (Pie Chart)

---

### 🔹 5️⃣ Dashboard Sheet
The main interactive view — a clean, consolidated layout showing all KPIs, charts, and slicers in one place.
